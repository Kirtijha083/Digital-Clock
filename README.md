# Digital Clock
This project is a Digital Clock built using HTML, CSS, and JavaScript. It displays the current time, including hours, minutes, seconds, and the AM/PM period.

# Features:
1. Displays time in a 12-hour format.
2. Shows the current time with leading zeros when necessary.
3. Automatically updates every second.
4. Displays AM/PM based on the time of day.

# How It Works:
1. HTML: Provides the structure for displaying the clock.
2. CSS: Used to style the clock elements.
3. JavaScript: Handles the dynamic behavior of fetching the current time and updating the clock every second.

# How to Run:
1. Clone or download the project files.
2. Open the index.html file in a web browser to see the clock in action.
